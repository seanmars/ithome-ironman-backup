### 為什麼需要觀察筆記？

牛郎工程師必備的秘密武器，就是**觀察筆記**  
你需要透過紀錄客人的**口頭禪偏好習慣**，累積成屬於自己的客人小檔案  
這不只是紀錄，而是為了在關鍵時刻，說出那句能打中人心的：你上次有提到的我幫你注意到了這種貼心細膩的情緒價值，會瞬間拉近你和客人的距離，讓客人覺得 被重視被理解

### 如何建立客人小檔案？

* 工具建議：HackMDEvernoteNotionExcelObsidian 都可以，重點是方便查閱和更新

  (示意圖)

#### 舉例：

* 目標：客人 A
  + 家庭：已婚有小孩住台北南部人
  + 興趣：爬山腳踏車羽球小說收藏鞋子
  + 投資偏好：數位貨幣美股
  + 辦公室習慣：~~早上泡咖再跑去跟商務部門女生聊天~~桌上喜歡小擺飾
  + 很想拿到這次Ｃ專案的管理權但又不想管事
  + 想知道Ｂ部門的內部消息
  + 很討厭Ｂ部門的另外一位客人
  + 飲食偏好：抹茶拿鐵雞排甜點蛋糕

#### 記得

* 觀察客人不是只靠聊天，而是每次會議對話細節中 慢慢累積定期更新筆記，包括：
  + 決策方式
  + 開會模式
  + 容易逃避的問題
  + 臨時興趣的變化

### 情境應用

* 客人現在只關注 C 專案，對 AB 專案幾乎不上心，進度停擺也不追但換個角度想這意味著負責 AB 的人暫時可以輕鬆一些，晨會時打打馬虎就能混過一天
* 客人早上固定會花時間泡咖啡閒聊這段時間幾乎不會談公事，剛好是跑出去買早餐的最佳時機
* 你知道客人雖然掛著技術主管的頭銜，但其實對技術細節並不熟悉一旦會議中出現技術問題，他往往會選擇閃躲這時候不用硬聊細節，反而可以順勢拋出一個更高層次的議題，例如：其實這部分應該朝 ×× 方向處理最後再補一句：你也是這樣想的吧？- 通常對方就只能順著回答對...雖然最後問題還是要自己解決，但至少這場無效會議能快速收尾，主管也會覺得你有方向有主導權，心裡對你的信任又多了一分（這招相當實用，推薦大家嘗試）

記住這些筆記並不應該只是存在資料夾裡，而是要被轉化為你日常工作的絕佳利器
